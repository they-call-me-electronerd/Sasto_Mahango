# MulyaSuchi (मूल्यसूची)

MulyaSuchi is a Nepal Market Intelligence Platform designed to provide accurate and up-to-date market prices for various commodities.

## Project Structure

The project is organized as follows:

- **`admin/`**: Admin panel for managing users, items, and validations.
- **`assets/`**: Static assets including CSS, JavaScript, images, and uploads.
- **`backups/`**: Backup files and restored versions of scripts.
- **`classes/`**: Core PHP classes (Database, Auth, User, Item, etc.).
- **`config/`**: Configuration files (Database, Constants, etc.).
- **`contributor/`**: Contributor panel for submitting price updates.
- **`docs/`**: Project documentation, guides, and implementation plans.
- **`includes/`**: Reusable PHP snippets (Header, Footer, Functions).
- **`public/`**: Public-facing pages (Home, Products, About, etc.).
- **`sql/`**: Database SQL scripts.
- **`tests/`**: Test scripts and debugging tools.

## Getting Started

1.  Configure your database settings in `config/database.php`.
2.  Import the SQL schema from `sql/`.
3.  Navigate to `public/index.php` to view the site.
